Programmeren

Voor het programmeren gebruik ik onderandere de p5.js web editor (JavaScript), Unity (C#) en de Visual Studio Code (HTML, JavaScript, Python, etc.).

De p5.js web browser gebruik ik vooral om te puzzelen met JavaScript. Op het eerste plaatje hieronder zie je hoe p5.js er uit ziet.
Visual Studio Code gebruik ik vooral voor dingen zoals websites zoals deze maken. Met dit programma kun je met verschillende programmeer talen werken zoals HTML, JavaScript, Python, etc. Bijna elke taal werk hiermee.
Unity is een ander soort programma. Deze is speciaal bedoelt voor 3D. Hier heb je dus al een renderer, belichting, een input-systeem, krachter, etc. Je moet alleen zelf alles samen laten werken met je C# code. Wanneer je Unity installeert krijg je automatisch Visual Studio er bij, Visual Studio Code is een variant hiervan.

Project 1: Databases en SQL

Bij dit project hebben we geleerd over MySQL. Hiermee kun data opvragen uit een database met behulp van query's. Hier leerden we vooral de basis van MySQL, zoals hoe je een query opstelt. In het gedeelte "verdieping" werdt dieper in gegaan op sommige concepten zoasl groeperen en subquery's.

Project 2: Datastructuren

Het eerste project dat we dit jaar hadden had te maken het datastructuren in Python. Hier leerden wij de verschillen tussen onderandere lijsten, queue's en stacks. Ook zijn de multidimentionale lijst en binary tree and de hand gekomen.
